# 🔱 Neptune CRM (海王控制台) v13.0

> **"Code Your Social Dynamics. Predict The Future."**
> **“将社交动态代码化，让不确定性坍缩。”**

Neptune CRM 是一个基于 **[第一性原理]** 和 **[控制论]** 构建的本地化单页应用 (SPA)。它不仅仅是一个通讯录，它是你的**外置社交大脑**，也是你的**战术指挥中心**。

## 核心哲学 (Core Philosophy)

* **数据本地化 (Local First)：** 所有数据存储在浏览器 LocalStorage，拒绝云端泄露，绝对的隐私主权。
* **反脆弱架构 (Anti-Fragile)：** 极简代码，无外部依赖，单文件运行，永不宕机。
* **AI 增强 (AI Augmented)：** 接入 Google Gemini 2.5 Flash 模型，提供战术级实时决策支持。

## ⚡ 核心功能 (Features)

### 1. 🎯 鱼塘看板 (The Dashboard)
* **全景视图：** 抛弃线性列表，采用赛博朋克风格的仪表盘。
* **状态追踪：** 实时监控目标状态（`刚认识` / `暧昧` / `邀约` / `已拿下`）。
* **冷却算法：** 自动计算上次联系时间。
    * ❄️ **冷冻中 (<3h)：** 防止需求感暴露。
    * ⚠️ **需维护 (>24h)：** 红色高亮预警，防止关系熵增。

### 2. 🧠 双重档案系统 (Dual-Memory System)
为了解决“隐私”与“AI分析”的矛盾，我们设计了物理隔离的档案结构：
* **🟩 战术情报 (Tactical Intel)：** 发送给 AI 的脱敏数据（如：性格、爱好、当前局势）。
* **🟧 绝密档案 (Top Secret)：** **本地锁死**，严禁联网（如：真名、住址、私密备注）。

### 3. 🤖 AI 战术指挥官 (AI Commander)
不仅仅是聊天机器人，而是内置了顶级社交心法的**军师**。
支持两种核心战术模式：
* **😈 浪子模式 (Bad Boy)：** * *核心：* 无所谓 (Outcome Independence) + 推拉 (Push-Pull)。
    * *场景：* 破冰、升温、对抗废物测试。
* **💎 奖品模式 (Prize)：** * *核心：* 强框架 (Frame Control) + 筛选 (Screening)。
    * *场景：* 确立高价值地位，逆转追逐关系。

### 4. ⚔️ 战术军火库 (Tactical Arsenal)
左侧边栏预设经典话术模板，点击即复制：
* `[打压 Neg]`
* `[推拉 Push/Pull]`
* `[赋格 Framing]`
* `[冷读 Cold Read]`
* `[邀约 Close]`

## 🚀 快速开始 (Quick Start)

1.  直接访问在线版本：https://yuli9999.github.io/Neptune-CRM/
2.  或者下载本项目，双击 `index.html` 即可运行。
3.  点击左下角 `[配置]`，填入你的 Google Gemini API Key（在google AI Studio免费申请）。
警告:一定要买正规API密钥  不要在淘宝闲鱼拼多多买第三方中转API密钥(你的所有数据都会存在他们数据库上)
4.  开始你的game
